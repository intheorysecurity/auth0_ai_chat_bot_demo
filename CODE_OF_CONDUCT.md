# Code of conduct

We want a respectful, harassment-free experience for everyone.

- Be constructive in reviews and discussions.
- Accept feedback gracefully; assume good intent.
- Do not share private or identifying information without consent.

For the full standard, see the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

**Enforcement:** report concerns to repository maintainers via a private channel (e.g. GitHub Security or maintainer email).
